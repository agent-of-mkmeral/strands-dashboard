ℹ️ How Mentions Work

The mentions.yml workflow polls GitHub notifications hourly (8am–10pm UTC) for @agent-of-mkmeral mentions.

Flow: Mention detected → Author authorization check → agent.yml dispatched → Agent responds

Security: Only mentions from authorized users trigger the agent. Unauthorized mentions are silently ignored.
